    Social Proof Section

A responsive Social Proof Section built with HTML and CSS. This project is a solution to the Frontend Mentor Social Proof Section challenge. It showcases customer ratings and testimonials in a clean, modern, and responsive layout.



    About the Project

The Social Proof Section project was created to strengthen my HTML and CSS skills by recreating a real-world UI design from Frontend Mentor. The project focuses on responsive layouts, semantic HTML, CSS Flexbox, typography, and spacing to closely match the original design.


     Features

- Responsive layout
- Customer rating cards
- Modern and clean user interface
- Semantic HTML structure
- CSS Flexbox layout
- Mobile-friendly design



  Technologies Used

- HTML
- CSS
